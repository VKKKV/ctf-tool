Cache les octets de data dans pixels selon l\textquotesingle{}algorithme de protection des données en L\+SB. 

Pour l\textquotesingle{}insertion, à partir du mot de passe, est créé un seed. Puis les octets contenus dans data seront mélangés dans des pixels aléatoires. Pour l\textquotesingle{}extraction, la seed va permettre de remettre dans le bon ordre les octets mélangés.

\hline

\hline
