◆ mp3_id3v1_tag_seek()

int mp3_id3v1_tag_seek ( FILE *  f)

Saute le tag ID3v1 actuel.

Paramètres
fFichier MP3.
Renvoie
Valeur de retour de fseek().
Prérequis
Le fichier f doit être ouvert en lecture. Lors du déplacement, il est considéré que la signature du tag vient d'être lu, le curseur de lecture est donc à "debut_du_tag + sizeof(sig)".
Effet de bord
Le curseur de lecture du fichier est modifié pour sauter le tag.
Auteur
Pierre Ayoub, Damien Delaunay

◆ mp3_id3v1_tag_write()

int mp3_id3v1_tag_write ( FILE *  src,
FILE *  dst 
)

Écris le tag ID3v1 actuel.

Paramètres
srcFichier MP3 source.
dstFichier de destination.
Renvoie
-1 sur une erreur, sinon 0.
Prérequis
Les fichiers src et dst doivent êtres ouverts en lecture. Lors de l'écriture, il est considéré que le header du tag à déjà été lu puis écrit, les curseurs de lecture et d'écriture sont donc à "debut_du_tag + sizeof(hdr)".
Effet de bord
Les curseurs de lecture du fichier src et d'écriture du fichier dst sont modifiés pour lire puis écrire la frame.
