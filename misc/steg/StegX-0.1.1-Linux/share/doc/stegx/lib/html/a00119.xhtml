Algorithme de protection des données avec le mot de passe.

◆ protect_data()

int protect_data ( uint8_t *  tab,
uint32_t  hidden_length,
const char *  passwd,
mode_e  mode 
)

Fait le mélange ou réarrange les octets selon l'algorithme de protection des données.

Pour l'insertion, à partir du mot de passe, est créé un seed. Puis les octets contenus dans tab seront mélangés. Pour l'extraction, la seed va permettre de remettre dans le bon ordre les octets mélangés.

Paramètres
tabTableau d'octets à mélanger.
hidden_lengthTaille du tableau tab (correspond à la taille du fichier à cacher/à analyser).
passwdMot de passe à partir duquel un seed sera créé pour la suite pseudo aleatoire nécessaire au mélange des octets de tab.
modeMode qui peut être {STEGX_STEGX_MODE_INSERT} ou {STEGX_MODE_EXTRACT}.
Renvoie
0 si le melange des donnees s'est bien passé ; 1 sinon.
Auteur
Clément Caumes
