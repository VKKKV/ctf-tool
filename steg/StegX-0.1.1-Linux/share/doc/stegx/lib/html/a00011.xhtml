Documentation du type de l'énumération

◆ err_code

enum err_code
Erreurs:
Liste des codes d'erreurs de la bibliothèque. Utilisé dans le retour des fonctions et dans la correspondance erreur <=> message. Le code 0 signifie (sauf cas particulier) qu'aucune erreur ne s'est produite.
Auteur
Pierre Ayoub
