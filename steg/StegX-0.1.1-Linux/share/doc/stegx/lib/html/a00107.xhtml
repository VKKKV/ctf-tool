◆ stegx_init()

info_s* stegx_init ( stegx_choices_s *  choices)

Initialise la bibliothèque.

Initialise la bibliothèque avec les choix de l'utilisateur en remplissant les structures nécéssaire et en ouvrant les fichiers.

Effet de bord
Initialise la structure info_s en fonction des choix de l'utilisateur. Il restera les champs info::hidden_length, info::algo, info::host::type, info.host.file_info à initialiser.
Prérequis
La structure stegx_choices_s doit être initialisée comme indiquée dans sa description.
Erreurs:

ERR_HOST si une erreur survient pendant l'ouverture du fichier hôte.

ERR_HIDDEN si une erreur survient pendant l'ouverture du fichier à caché.

ERR_RES_INSERT si une erreur surivient pendant l'ouverture du fichier résultat lors de l'insertion.

ERR_RES_EXTRACT si une erreur surivient pendant l'ouverture du dossier résultat lors de l'extraction.

ERR_PASSWD si le mot de passe fourni est non-conforme.

Paramètres
choicesStructure contenant les choix de l'utilisateur.
Renvoie
Pointeur sur la structure privée qui contient les informations pour réaliser la dissimulation et l'extraction, sinon NULL sur une erreur et met à jour la variable stegx_errno.
Auteur
Clément Caumes et Pierre Ayoub
