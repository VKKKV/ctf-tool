◆ read_signature()

static int read_signature ( info_s *  infos)
static

Lit la signature contenu dans le fichier hôte.

Effet de bord
Renseigne la structure infos_s avec les informations contenues dans la signature.
Erreurs:
ERR_NEED_PASSWD si le récepteur n'as pas fourni de mot de passe alors qu'il aurait dû.
Paramètres
infosStructure représentant les informations concernant l'extraction.
Renvoie
0 si la signature a bien été lue, sinon 1 et assigne stegx_errno à l'erreur survenue.
Auteur
Pierre Ayoub et Damien Delaunay

◆ stegx_detect_algo()

int stegx_detect_algo ( info_s *  infos)

Detecte l'algorithme utilisé ainsi que le nom du fichier caché et la taille des données cachées.

Effet de bord
Remplit les champs info_s.host.file_info, info_s::algo, info_s::method, info_s::hidden_length, info_s::hidden_name.
Paramètres
infosStructure représentant les informations concernant la dissimulation.
Erreurs:

ERR_DETECT_ALGOS si une erreur quelconque survient.

ERR_NEED_PASSWD si l'utilisateur n'as pas spécifié le mot de passe requis.

Renvoie
0 si la détection s'est bien déroulée, sinon 1 en cas d'erreur et met à jour stegx_errno.
Auteur
Pierre Ayoub et Damien Delaunay
