Codes d'erreurs
Global(e) err_code
Liste des codes d'erreurs de la bibliothèque. Utilisé dans le retour des fonctions et dans la correspondance erreur <=> message. Le code 0 signifie (sauf cas particulier) qu'aucune erreur ne s'est produite.
Global(e) fill_host_info (info_s *infos)

ERR_SUGG_ALGOS si la fonction est utilisé en mode extraction.

ERR_CHOICE_ALGO si l'algorithme choisi n'est pas disponible.

Global(e) read_signature (info_s *infos)
ERR_NEED_PASSWD si le récepteur n'as pas fourni de mot de passe alors qu'il aurait dû.
Global(e) stegx_check_compatibility (info_s *infos)
ERR_CHECK_COMPAT si la vérification de la compatibilité a échouée.
Global(e) stegx_choose_algo (info_s *infos, algo_e algo_choosen)

ERR_SUGG_ALGOS si la fonction est mal utilisée (mauvais mode).

ERR_CHOICE_ALGO si l'algorithme choisi n'est pas disponible.

Global(e) stegx_detect_algo (info_s *infos)

ERR_DETECT_ALGOS si une erreur quelconque survient.

ERR_NEED_PASSWD si l'utilisateur n'as pas spécifié le mot de passe requis.

Global(e) stegx_extract (info_s *infos, char *res_path)
ERR_EXTRACT si une erreur survient lors de l'extraction.
Global(e) stegx_init (stegx_choices_s *choices)

ERR_HOST si une erreur survient pendant l'ouverture du fichier hôte.

ERR_HIDDEN si une erreur survient pendant l'ouverture du fichier à caché.

ERR_RES_INSERT si une erreur surivient pendant l'ouverture du fichier résultat lors de l'insertion.

ERR_RES_EXTRACT si une erreur surivient pendant l'ouverture du dossier résultat lors de l'extraction.

ERR_PASSWD si le mot de passe fourni est non-conforme.

Global(e) stegx_insert (info_s *infos)
ERR_INSERT si une erreur survient durant la dissimulation.
Global(e) stegx_suggest_algo (info_s *infos)

ERR_SUGG_ALGOS s'il y a eu un problème d'utilisation.

ERR_LENGTH_HIDDEN si le fichier à cacher est supérieur à la taille maximum (2^32).
