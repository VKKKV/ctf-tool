◆ protect_data_lsb()

int protect_data_lsb ( uint8_t *  pixels,
uint32_t  pixels_length,
uint8_t *  data,
uint32_t  data_length,
char *  passwd,
mode_e  mode 
)

Cache les octets de data dans pixels selon l'algorithme de protection des données en LSB.

Pour l'insertion, à partir du mot de passe, est créé un seed. Puis les octets contenus dans data seront mélangés dans des pixels aléatoires. Pour l'extraction, la seed va permettre de remettre dans le bon ordre les octets mélangés.

Paramètres
pixelsTableau d'octets representant les pixels.
pixels_lengthTaille du tableau pixels (correspond à la taille du bloc data).
dataTableau d'octets representant les donnees a cacher.
data_lengthTaille du tableau data (correspond à la taille des données a cacher).
passwdMot de passe à partir duquel un seed sera créé pour la suite pseudo aleatoire nécessaire au mélange des octets de tab.
modeMode qui peut être {STEGX_STEGX_MODE_INSERT} ou {STEGX_MODE_EXTRACT}.
Renvoie
0 si le melange des donnees s'est bien passé ; 1 sinon.
Auteur
Clement Caumes
